\subsection{Detailed Description}
Handle the Sparse matrix kernels. This module implements the library's support kernels which perform error-\/checking first and then calls the appropriate kernel implementation.

\begin{DoxyAttention}{Attention}
Currently, only support Ax and A$^\wedge$\-Tx kernels.
\end{DoxyAttention}
